1. Introduction to CSS

CSS (Cascading Style Sheets) styles HTML elements. It adds colors, typography, layouts, and more.

HTML
<p style="color: red;">Red Text</p>
CSS
p { color: blue; font-size: 16px; }
Exercise: Create a paragraph and style it red using inline CSS.

2. CSS Selectors

CSS

/* Element Selector */ 
p { color: purple; }

/* Class Selector */ 
.highlight { background-color: yellow; }

/* ID Selector */ 
#main { font-size: 20px; }

/* Group Selector */ 
h1, h2, h3 { font-family: Arial; }
Exercise: Create an <h1> and <p> and style using ID and class.

4. CSS Flexbox

HTML

<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
CSS

.container {
  display: flex;
  justify-content: space-around;
  align-items: center;
}
.box {
  width: 50px;
  height: 50px;
  background: teal;
  color: white;
  text-align: center;
}
Exercise: Create 4 boxes and use justify-content: space-between;

5. Mini Project: Profile Card

HTML

<div class="profile-card">
  <img src="profile.jpg" class="profile-img">
  <h2>John Doe</h2>
  <p>Software Engineer</p>
  <button>Follow</button>
</div>
CSS

.profile-card {
  width: 250px;
  padding: 20px;
  border: 1px solid #ccc;
  border-radius: 10px;
  text-align: center;
  box-shadow: 2px 2px 10px rgba(0,0,0,0.1);
}
.profile-img {
  width: 100px;
  border-radius: 50%;
}
button {
  padding: 10px 20px;
  background-color: blue;
  color: white;
  border: none;
  border-radius: 5px;
  cursor: pointer;
}
Exercise: Add hover effect to button and change card background color.
